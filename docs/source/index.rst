

CMEPDA-EXAM Documentation
==========================

Welcome to the official documentation for CMEPDA-EXAM, a comprehensive software suite designed for medical image classification and analysis. This documentation provides an overview of the available tools, usage instructions, and guidance on running machine learning classifiers as well as convolutional neural networks (CNNs) included in the project.


Contents:
---------------
This documentation provides a comprehensive guide to the CMEPDA-EXAM project, including setup instructions, detailed descriptions of datasets and key components, usage guidelines for both machine learning and CNN models, and contribution guidelines to help you get involved.
